Clarity

Extension settings

Clarity is open source. Run the backend from this folder or point the extension at your own deployment. The AI key lives in your backend .env, not in the extension.

Backend URL

Use http://localhost:3000 for the local server in this repo, or enter your own self-hosted base URL.

Clarity calls /api/explain and /api/health on this server.

Trigger shortcut

Highlight text first, then press this shortcut to open Clarity.
