# Clarity.AI Launch Guide

Use this as the final pass before publishing the repo or sharing it publicly.

## Suggested one-line description

Open-source browser extension that explains highlighted text in place with Gemini.

## Suggested screenshots

1. A normal webpage with a highlight and the Clarity.AI popup open.
2. The popup showing `Explain like I'm 12` or `Explain deeply`.
3. The settings page in `Direct Gemini (no server)` mode.
4. The standalone result page from the context-menu flow.
5. Optional: the local backend health page for self-hosting users.

## 30-second demo flow

1. Highlight a phrase on a real article or docs page.
2. Trigger Clarity.AI with the shortcut.
3. Show the first explanation.
4. Click `Explain like I'm 12`.
5. Copy the explanation.
6. Open settings and briefly show direct Gemini mode.

## Before publishing

1. Confirm `.env` is not tracked.
2. Reload the unpacked extension.
3. Test direct Gemini mode once end to end.
4. Test backend mode if you plan to document it.
5. Add at least one clean screenshot to the GitHub repo or release post.
